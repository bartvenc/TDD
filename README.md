# Installation
## Dependencies:
- SDL2
- Boost 
- CLANG

### Ubuntu
```bash
$ sudo apt install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev clang-format-8
```

### OSX

```bash
$ brew install sdl2 sdl2_image sdl2_ttf boost clang-format
```

## Compile
> Cmake incomming....

```bash
$ make
```
